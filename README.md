# 🎮 TESANA: Clash of Warriors - Battle Royale Game

A full-stack multiplayer battle royale game inspired by PUBG with 100 players competing on a dynamic island.

## 📋 Project Structure

```
tesana-battle-royale/
├── server/                 # Backend (Node.js + Express + WebSocket)
├── client/                 # Frontend (React + Three.js)
├── game-engine/            # Game logic & physics
├── database/               # MongoDB schemas
├── assets/                 # Game assets & maps
├── config/                 # Configuration files
└── docs/                   # Documentation
```

## 🎯 Core Features

- ✅ 100 players battle royale matches
- ✅ Dynamic map with 5 zones
- ✅ 50+ weapons with realistic physics
- ✅ 6 character classes with unique abilities
- ✅ Real-time multiplayer using WebSockets
- ✅ Progression system & battle pass
- ✅ In-game shop & cosmetics
- ✅ Leaderboards & matchmaking

## 🚀 Quick Start

### Prerequisites
- Node.js 18+
- MongoDB Atlas
- React 18+

### Installation

```bash
# Clone repository
git clone https://github.com/warismeer7532-blip/tesana-battle-royale.git
cd tesana-battle-royale

# Install dependencies
npm install

# Setup environment
cp .env.example .env

# Start development
npm run dev
```
